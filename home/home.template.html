About

This application is a Single-page application(SPA) Which means this web site that interacts with the user by dynamically rewriting the current page rather than loading entire new pages from a server.

The page does not reload at any point in the process, nor does control transfer to another page

Three different kind of catalog(music, movie, phone) used one template but diffrent data-modle.Each catalog item's detail used diffrent template and data-modle

Tools:HTML/CSS, JavaScript, Bootstrap, AngularJS, Linux, Apache, AWS EC2
